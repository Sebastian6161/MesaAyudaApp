# Sistema de Mesa de Ayuda - Java

Proyecto desarrollado en Java utilizando Programación Orientada a Objetos (POO) para simular un sistema básico de Mesa de Ayuda y gestión de incidencias.

## Funcionalidades

✔ Crear tickets  

✔ Ver tickets registrados  

✔ Buscar tickets por ID  

✔ Cerrar tickets  

✔ Gestión de estados (Abierto/Cerrado)

## Tecnologías utilizadas

- Java

- IntelliJ IDEA

- Git

- GitHub
